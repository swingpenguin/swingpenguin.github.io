Ant Design 简单五步修改主题颜色

本文基于用 npm 作为包管理工具以及用 create-react-app 创建的 react app.

Step 1: 安装 less 和 less-loader

注意版本, 其他版本会报错/配置不成功

Step 2: Git commit unsaved changes

如果在用 git 的话先保存进度, 不然不能进行下一步的 eject

Step 3: 把 webpack 配置 eject 出来

如果 webpack 配置没有被封装的话不需要 eject, 可以跳过这一步, 直接到第四步配置

用这种方法的前提是安装了 react-script

Step 4: 修改配置文件

Eject 之后会出现 config 文件夹, 从里面找的 webpack.config.js 文件进行配置
在 module - rules 里面(plugs 之前)加入如下配置
在大约 540 行的位置, 可以看到 rules 里有很多格式相同的配置, 仿照其他的层次和位置就好.
这一步主要是加入 less-loader 使得 webpack 可以将 less 形式的文件编译并打包为浏览器可直接处理的形式.

这个配置参考 antd 文档定制主题 - Ant Design, 但是文档中有部分被 comment 掉了不知道为什么…
在 modifyVars -> ‘primary-color’里设置你喜欢的主题颜色即可!
还可以修改其他配置, 具体变量名请参考 Ant Design 文档.

Step 5: 修改引入样式文件.css to .less

没有生效的话参考 antd 文档定制主题 - Ant Design
“如果你是通过’antd_dist_antd.css’#引入样式的, 改为 antd_dist_antd.less”
一般是在 index 页面 import antd.
